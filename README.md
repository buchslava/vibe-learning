# Vibe Learning

A learning hub — practical books and exercises with **clear mental models**, not filler. Start from zero with **Rust from the Scratch**, or dive into **Rust Core** if you already program.

## Sub-books

| Book | Path | Focus |
|------|------|--------|
| **Rust from the Scratch** | [`rust-from-the-scratch/`](rust-from-the-scratch/) | Zero experience → games → local Snake; Trust & Use, playground-first |
| **Rust Core** | [`rust-core/`](rust-core/) | Language fundamentals: ownership, types, traits, concurrency, std I/O |
| *(more coming)* | — | Future tracks under this umbrella |

## Start here

**New to programming?** Open **[Rust from the Scratch → Preface](rust-from-the-scratch/chapters/preface.md)**, then **[README](rust-from-the-scratch/README.md)** and **[CONTENTS.md](rust-from-the-scratch/CONTENTS.md)**.

**Already code in another language?** Open **[Rust Core → Preface](rust-core/chapters/preface.md)**, then **[README](rust-core/README.md)** and **[CONTENTS.md](rust-core/CONTENTS.md)**.

Rust Core chapters include playground examples and **Afterparty** prompts for AI-assisted practice — see [AI Prompt Index](rust-core/appendix/AI_PROMPT_INDEX.md).
